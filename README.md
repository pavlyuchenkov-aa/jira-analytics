# JIRA Analytics Tool

Инструмент для аналитики данных из JIRA с визуализацией метрик проекта.

## Возможности

- Загрузка задач из JIRA API
- Анализ времени выполнения задач
- Визуализация метрик проекта
- Статистика по пользователям
- Распределение задач по приоритетам

## Установка

### Установка из исходного кода

```bash
# Клонируйте репозиторий
git clone https://github.com/pavlyuchenkov-aa/jira-analytics.git
cd jira-analytics

# Установите зависимости
pip install -r requirements.txt

# Установите пакет
pip install -e .
